    Zadatak

    GitHub app

    Potrebno je izraditi aplikaciju koja omogućava pretragu repozitorija i korisnika na GitHub servisu.

    GitHub ima vlastiti API čija se dokumentacija može pronaći na sljedećem linku: https://developer.github.com/v3/

    Popis endpoint-a u JSON formatu dostupan je ovdje: https://api.github.com/

    Aplikacija se sastoji od nekoliko ekrana čiji su zahtjevi opisani u daljnjem tekstu.

    Search screen

    Funkcionalnosti:

    -Prikaz glavnog ekrana aplikacije pri samom ulasku u aplikaciju (main screen)

    -Mogućnost unosa pojma za pretragu (query)

    -Prikaz liste rezultata za uneseni pojam pretrage

    -Mogućnost promjene sortiranja rezultata pri čemu rezultati mogu biti sortirani prema:

    ○Stars

    ○Forks

    ○Updated

    -Dodatne informacije o parametrima mogu se pronaći ovdje: https://developer.github.com/v3/search/#search-repositories

    Svaki element liste (rezultat) mora biti predstavljen sa sljedećim podacima (minimalno):

    -Ime repozitorija

    -Ime autora

    -Thumbnail slika autora

    -Broj pratitelja (watchers)

    -Broj fork-ova

    -Broj issue-a

    Po želji, za svaki rezultat možeš dodati i dodatne informacije ukoliko ih smatraš praktičnim ili potrebnim.

    Element liste mora omogućiti navigaciju​ na detalje repozitorija (​Repository detail screen) - npr. klikom na element liste

    Repository detail screen

    Funkcionalnosti:

    -prikaz detalja o jednom repozitoriju u zasebnom screenu

    -prikaz proširenog seta informacija u odnosu na informacije koje su dostupne na ekranu za pretragu. Odluči koje ćeš sve dodatne informacije prikazati na ovom ekranu. Npr. neke od korisnih informacija su:

    ○Ime programskog jezika

    ○Datum kreiranja

    ○Datum izmjene

    -Mogućnost otvaranja (navigacije) detalja repozitorija u eksternom browseru (npr. Chrome, Firefox, Safari)

    -prikaz informacija o vlasniku (owner-u)
